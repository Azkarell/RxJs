Was ist RxJs?

  • Ein Programm wird als Fluss/Strom (stream) von Daten angesehen
    Folge von:
    • Events
    • Http-Requests
    • Zuständen (States)
  • Um ein Programm schreiben zu können muss man diese Ströme verändern können
  • Es entstand das Konzept der Observable welche Datenstrom und Operationen vereint
  • RxJs implementiert dieses Konzept
  • RxJs steht für Reactive Extensions for Javascript
