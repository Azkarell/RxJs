Was ist RxJs?

  • Ein Programm wird als Fluss (stream) von Daten angesehen
    Folge von:
    • Events
    • HttpRequest
    • etc.
  • Um ein Programm schreiben zu können muss man diese Ströme verändern können
  • Es entstand das Konzept der Observable welche Datenstrom und Operationen auf diesen vereint
  • RxJs implementiert dieses Konzept welches in Angluar oft verwendet wird
  • RxJs steht für Reactive Extensions for Javascript
