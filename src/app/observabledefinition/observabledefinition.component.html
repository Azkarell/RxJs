Observable

  • Blueprint von Operationen die auf einen Stream aufgeruft werden sollen. 
    var observable = Observable.timer(0,1000).take(5)
  • Erst subscribe führt Code aus 
    var subscription = obs.subscribe()
  • Pro subscribe wird ein neuer Stream (soweit nicht anders verlangt) erstellt 
    var subscription = obs.subscribe(callback_fun)
    var subscription2 = obs.subscribe(callback_fun)
    sind zwei verschiedene unabhängige Subscriptions
